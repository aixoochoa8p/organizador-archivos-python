# Organizador Automático de Archivos

Este proyecto es un script en Python que organiza automáticamente los archivos de una carpeta según su extensión.  
Crea carpetas para cada tipo de archivo (PDF, imágenes, Excel, etc.) y mueve los archivos a su ubicación correspondiente.

## Funcionalidades
- Clasificación automática por extensión  
- Creación de carpetas según el tipo de archivo  
- Organización rápida y sin intervención manual  
- Compatible con cualquier carpeta local  

## Uso
```python
organizar_archivos("C:/ruta/de/tu/carpeta")
